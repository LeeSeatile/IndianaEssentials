# 💎 Indiana Jewels — E-Commerce Website

A chic, responsive front-end website designed for **Indiana Jewels**, a luxury jewelry brand focused on celebrating individual beauty through subtle, high-end pieces.

> *This project holds special significance as my very first web development project, marking the start of my journey in software engineering and web design.*

---

## 🌟 Overview

Indiana Jewels offers curated, high-quality jewelry designed to complement everyday style with elegance. This project implements a fully structured front-end interface built from scratch using clean HTML5 and custom CSS3.

### Key Features
* **Interactive Best Sellers Gallery:** Built using CSS Flexbox with custom hover transitions for dynamic product previews.
* **Responsive Layout:** Engineered with fluid meta-viewport settings and scalable CSS units for cross-device compatibility.
* **Custom Inquiry System:** Structured contact form supporting mass-produced and custom design requests.
* **Consistent Brand Styling:** Warm, luxury-inspired color palette (`#fbe8ca`, `#462415`, `#cb8c51`) paired with elegant typography choices.

---

## 🛠️ Tech Stack

* **Language:** HTML5
* **Styling:** Custom CSS3 (Flexbox, CSS Transitions, Custom Palette)
