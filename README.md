# Indiana Essentials — E-Commerce & Product Distribution Platform

## 📌 Problem Statement
Small-to-medium retail and distribution ventures often struggle with fragmented product showcases, poor user interface design, and friction during customer catalog navigation, leading to lower user engagement and missed conversion opportunities.

## 💡 Solution & Features
**Indiana Essentials** is a responsive web application designed to streamline product presentation, catalog navigation, and consumer interaction.
* **Interactive Product Catalog:** Intuitive product layout with category filtering and clear item showcase cards.
* **Responsive UI Design:** Fully adaptable layout optimized for desktop, tablet, and mobile viewing.
* **Streamlined User Experience:** Clean navigation paths designed to simplify customer engagement and product inquiry workflows.

## 🛠️ Tech Stack
* **Front-End:** HTML5, CSS3, JavaScript
* **Design & Tools:** Visual Studio Code, Git, GitHub Desktop
